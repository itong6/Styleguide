GRAPHICS & ICONS

These illustrations maintain the minimal, cartoon-like aesthetic of Meaty. This particular style is used to captivate the audience with cute and friendly avatars, contrasting and accentuating the essential message that the application is trying to convey. Unsuspecting users will believe the Meaty process is a cheerful and wholesome simulation, however soon after they have launched Meaty, the app will take a graphics yet realistic turn of events, making the message all the more impactful.

Sprites

These are the main default sprites of the Meaty application. In the application, you are able to select variants of these animals, however these are the base designs. These illustrations may also act as unofficial mascots of Meaty.

Patterns

The Meaty application also features patterns. These patterns appear at the end of the Meaty process, where users are greeted with facts and information revolving around food-origin awareness. Depending on which animal selected, one of the three distinct patterns will be in the background of the interface.

Stunning Methods

The Meaty application also features stunning methods. These weapons appear during the stunning method selection process and the objective tasks, where users are given a task to complete. Depending on which stunning method, one of the four stunning methods will have a unique task for the user to complete.

Weapons

The Meaty application also features weapons. These weapons appear during objective tasks requiring the users to use them, where users are given a task to complete. The user will require to use both weapons to complete the game process with both weapons have a unique method of using them to complete the objective task.
