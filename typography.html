We are using two main fonts of the same typeface, Roboto Mono Medium and Roboto Mono Regular. Using these two fonts will keep our design consistent.

We chose Roboto Mono because it has the pixel game style that fitted with our design. The typeface would be used for instructions to our users for a step by step game.

Roboto Mono Medium: This font offers a clean bold but not bold version of Roboto Mono, it will be our main font for headers.

Roboto Mono Regular: This is the original version of Roboto Mono and will be used as our body text. Users will be reading this font throughout the game when reading the objective description.
